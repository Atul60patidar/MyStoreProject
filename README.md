# MyStore E-Commerce Automation Framework

## Project Overview
This project is developed using Selenium WebDriver, Java, TestNG, and Maven for automating an E-Commerce web application.

## Features
- Login Testing
- Product Search
- Add to Cart
- Checkout Process
- Screenshot Capture
- TestNG Reporting

## Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- GitHub

## Framework Design
- Page Object Model (POM)
